# 도로 세그멘테이션 비교 결과

## 방법별 특성

| 방법 | 장점 | 단점 | 적합 용도 |
|------|------|------|----------|
| 색상 기반 (HSV) | 빠름, 단순, 실시간 처리 가능 | 조명 변화에 민감, 그림자 영향 | 빠른 프로토타이핑, 제한된 환경 |
| 엣지 기반 (Hough) | 차선 검출 우수, 구조적 특징 강조 | 복잡한 장면 처리 어려움, 노이즈 민감 | 차선 검출, 도로 경계 추출 |
| SegFormer | 정확도 높음, 최신 트랜스포머 기반 | GPU 필수, 처리 속도 느림 | 정밀 세그멘테이션, 연구/분석 |
| SAM/GrabCut | 적응적, 프롬프트 기반 제어 | 초기화/프롬프트 필요, 복잡한 설정 | 대화형 세그멘테이션, 맞춤형 추출 |
| DeepLabV3 | 범용성, 다양한 객체 분할 | 도로 특화 안됨, 리소스 소모 | 일반 객체 분할, 다목적 용도 |

## 성능 비교

### 처리 속도
1. **색상 기반 (HSV)**: ~0.1초/이미지 (최고속)
2. **엣지 기반 (Hough)**: ~0.2초/이미지 (고속)
3. **DeepLabV3**: ~0.5초/이미지 (중간)
4. **SegFormer**: ~1.0초/이미지 (느림)
5. **SAM/GrabCut**: ~2.0초/이미지 (최저속)

### 정확도 (도로 세그멘테이션)
1. **SegFormer**: ★★★★★ (최고)
2. **DeepLabV3**: ★★★★☆
3. **SAM/GrabCut**: ★★★☆☆
4. **색상 기반**: ★★☆☆☆
5. **엣지 기반**: ★★☆☆☆ (차선에 특화)

### 적용 난이도
1. **색상 기반**: ★☆☆☆☆ (매우 쉬움)
2. **엣지 기반**: ★★☆☆☆ (쉬움)
3. **DeepLabV3**: ★★★☆☆ (보통)
4. **SegFormer**: ★★★★☆ (어려움)
5. **SAM/GrabCut**: ★★★★★ (복잡함)

## 결과 파일

### 비교 이미지
- `comparison_01CT000000164.png`: 도시 도로 비교
- `comparison_02CT000000181.png`: 고속도로 비교
- `comparison_03CT000000034.png`: 시골길 비교
- `comparison_04CT000001344.png`: 야간 도로 비교
- `comparison_05CT000000473.png`: 복잡한 교차로 비교

각 비교 이미지는 6개 패널로 구성:
1. Original: 원본 이미지
2. Color (HSV): HSV 색상 기반 세그멘테이션
3. Edge (Hough): Hough 변환 차선 검출
4. SegFormer: 트랜스포머 기반 세그멘테이션
5. SAM/GrabCut: Segment Anything Model + GrabCut
6. DeepLabV3: DeepLab v3+ 세그멘테이션

## 사용 방법

```bash
# 비교 이미지 생성
python results/comparison/create_comparison.py

# 결과 확인
ls results/comparison/comparison_*.png
```

## 권장 사항

### 실시간 애플리케이션 (자율주행 등)
- **1순위**: 색상 기반 (HSV) - 속도 우선
- **2순위**: 엣지 기반 (Hough) - 차선 검출

### 정확도 우선 (연구/분석)
- **1순위**: SegFormer - 최고 정확도
- **2순위**: DeepLabV3 - 범용성

### 대화형 도구
- **1순위**: SAM/GrabCut - 사용자 제어 가능

## 기술적 세부사항

### 하드웨어 요구사항
- **CPU Only**: 색상 기반, 엣지 기반
- **GPU 권장**: SegFormer, DeepLabV3
- **GPU 필수**: SAM (대용량 모델)

### 메모리 사용량
- **낮음 (~500MB)**: 색상 기반, 엣지 기반
- **중간 (~2GB)**: DeepLabV3
- **높음 (~4GB)**: SegFormer
- **매우 높음 (~8GB)**: SAM

## 결론

각 방법은 특정 사용 사례에 최적화되어 있습니다:
- **빠른 프로토타이핑**: 색상/엣지 기반
- **생산 환경**: SegFormer (정확도) 또는 색상 기반 (속도)
- **연구/분석**: SegFormer 또는 DeepLabV3
- **맞춤형 작업**: SAM/GrabCut

선택은 속도, 정확도, 리소스 간의 트레이드오프에 달려 있습니다.
